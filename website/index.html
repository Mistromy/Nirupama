// what it does

The main acts

Three things Nirupama takes seriously. Everything else is in the toolbox below.

@Nirupama

An AI with opinions

Mention the bot and talk. It keeps a personality you can tune per server — or let it adapt to your channel's vibe on its own. It can generate code, react to messages, and dig up GIFs mid-conversation.

/ship

Ship math with receipts

Most ship commands roll a random number. This one reads actual user data to compute compatibility, then writes you an AI-generated analysis of the result. Destiny, but auditable.

screenshot slot /ship result card

/graph

Activity, plotted

Message counts tracked per user over time, stored properly in Postgres, and rendered as a clean graph on demand. The groundwork for the leveling system that's on the way.

screenshot slot /graph output · e.g. graph.png

Also in the toolbox

🎲 dice rolls 🎱 magic 8-ball 🎭 tone analysis 💡 in-Discord suggestions ⚙️ tunable AI personality

One person, one bot

Nirupama is Mistromy's long-running playground — it started as a level bot storing data in a JSON file and kept evolving. It's where new ideas get tested, which means it changes often and improves constantly.

It's open source, so you can read exactly what it does. Found a bug or want a feature? /suggest works, and so do GitHub issues.

Running the AI costs actual money. If the bot earns its keep in your server, there's a Ko-fi. If not, that's fine too.
